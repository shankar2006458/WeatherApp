# 🌦️ Atmos Weather App

A premium modern weather dashboard built with **HTML, Tailwind CSS, and JavaScript**, powered by **Open-Meteo API**.

Live weather, hourly forecast, 7-day forecast, saved cities, search history, dynamic animated backgrounds, and a polished responsive UI for desktop + mobile.

🔗 https://shankar2006458.github.io/WeatherApp/

## ✨ Features

### 🌍 Weather Core
- Real-time current weather
- 24 Hour forecast
- 7 Day forecast
- Feels like temperature
- Wind speed
- Humidity
- Pressure
- UV Index
- Sunrise / Sunset

### 🔎 Smart Search
- Search any city worldwide
- Fast city autocomplete
- Recent search history
- Delete history items
- Add city directly from search

### ❤️ Saved Cities
- Save favorite cities
- Remove saved cities
- Auto open first saved city after refresh
- Default fallback city: Asansol

### 🎨 Premium UI
- Glassmorphism cards
- Dynamic weather backgrounds
- Animated hero section
- Smooth transitions
- Mobile bottom navigation
- Fully responsive layout

### 📱 Device Ready
- Desktop optimized
- Tablet optimized
- Mobile optimized
